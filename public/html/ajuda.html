Ajuda para aplicação de Telefonia VOIP 3CXPhone

Esta seção foi criada com o intuito de auxiliar os colaboradores na utilização do aplicativo 3CXPhone utilizado para chamadas telefonicas

Como utilizar o app

Para utilizar o aplicativo é necessário mante-lô sempre aberto para que esteja disponível para receber e realizar ligações

O aplicativo ficará aberto em segundo plano, desta forma para visualizar se está em funcionamento verifique no menu localizado na barra de tarefas como o da imagem

Ícone na barra de tarefas

Ícone no menu da barra de tarefas

Ao clicar no ícone do aplicativo o telefone será aberto na tela

Status

Como destacado na primeira imagem no canto inferior direito é mostrado o status do telefone, por padrão é definido Available que significa que o seu telefone está disponível para receber qualquer chamada

Além deste existem mais 3 status que podem ser utilizados, são eles: DND, Auto-Ans e Auto-VM

Para alterar o status basta clicar em cima

Status DND

O Status DND define o telefone disponível apenas para chamadas internas, ou seja, para ramais internos da empresa

Status Auto-Ans

O Status Auto-Ans define o telefone para atender automaticamente as chamadas

Status Auto-VM

O Status Auto-VM define o telefone para receber automaticamente a mensagem no correio eletronico

Ligações

Para fazer ligações basta discar o numero tanto no aplicativo quanto utilizando o teclado numerico do computador e apertar no botão verde ou a tecla enter

Para puxar ligações de ramais do mesmo setor basta discar *8

Para realizar chamadas para o DDD 011 basta discar 0 e o numero desejado

Ligações para demais DDD's deve ser discado 00 + DDD + numero de telefone

Discagem de ramal

Durante a ligação o aplicativo irá mostrar como na imagem acima

Quando estiver recebendo chamada será mostrado como na imagem acima

Botões

Os botões são dividos em 6 funcionalidades são elas:

Histórico de chamadas, Chamadas perdidas, Lista de contatos, Gravação de ligações, Numero para chamada, Mudo

Histórico de chamadas

Este botão irá mostrar as chamadas que foram realizadas pelo usuário

Ao clicar no botão é mostrado uma grade como na imagem abaixo com as chamadas realizadas indicando qual o número do ramal que estava em contato, o nome do contato e duração das chamadas

Histórico de Chamadas

As chamadas aparecerão como na imagem acima

Chamadas perdidas

Ao clicar no botão é mostrado uma lista com as chamadas perdidas com o contato e as informações de data e hora

Chamadas perdidas

A lista de chamadas perdidas aparecerão como na imagem acima

Lista de contatos

Ao clicar no botão é mostrado uma lista com os contatos salvos

Lista contatos

A lista de contatos aparecerá como na imagem acima

Botões para ligar, cadastrar novo, alterar, excluir contato, excluir toda listagem, este botão irá funcionar no contato indicado em amarelo

Para cadastrar novo contato basta preencher os dados: Nome para ser mostrado, primeiro e ultimo nome, email e numero

Gravação de chamadas

Ao clicar no botão é mostrado uma lista com as chamadas que foram gravadas, por padrão é desabilitado, caso deseje gravar basta habilitar os parametros clicando nas caixas

Gravar chamadas

A lista de chamadas irá aparecer na lista disponível

Mensagem eletronica

Ao clicar no botão é mostrado uma opção de configuração para correio eletronica para que seja deixado mensagens

Correio eletronica

Selecionar um numero para receber caixa postal
